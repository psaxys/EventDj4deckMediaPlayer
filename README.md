# EventDj4deckMediaPlayer
Нативное macOS-приложение на для эвент dj  предназначенное для удобного показа изображений и видео на втором экране. Проект поддерживает drag-and-drop медиа, 4 preview-слота, горячие клавиши, управление воспроизведением, loop/mute/autoplay, fallback-заставку, автопереключение дисплеев и сохранение конфигурации пользователя. Для macOS 13+
## Обновления

Приложение поддерживает ручную и автоматическую проверку обновлений через GitHub Releases.

### Как это работает
- При запуске приложение может автоматически проверять наличие новой версии.
- Пользователь также может запустить проверку вручную через меню **«Обновления»** или **«Настройки → Обновления»**.
- Если доступна новая версия, приложение показывает окно с информацией об обновлении.
- Из этого окна можно открыть страницу релиза или прямую ссылку на установочный файл (`.dmg`, `.pkg`, `.zip`), если он опубликован.

### Источник обновлений
Проверка выполняется по репозиторию:

`https://github.com/psaxys/EventDj4deckMediaPlayer`

с использованием GitHub Releases API.

### Важно
- Приложение не обновляет себя «поверх» автоматически.
- Механизм обновлений помогает найти и открыть актуальный релиз для скачивания и установки.
- Настройки проверки обновлений сохраняются в пользовательских параметрах приложения.
