# EventDj4deckMediaPlayer
Нативное macOS-приложение на для эвент dj  предназначенное для удобного показа изображений и видео на втором экране. Проект поддерживает drag-and-drop медиа, 4 preview-слота, горячие клавиши, управление воспроизведением, loop/mute/autoplay, fallback-заставку, автопереключение дисплеев и сохранение конфигурации пользователя. Для macOS 14+
